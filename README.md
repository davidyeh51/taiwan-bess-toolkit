# 用電大戶表後儲能 HTML 簡報工具包

這是一份可直接用 GitHub Pages 分享的靜態 HTML 簡報，內容包含：

- Part 1：用電大戶法規說明，整理現行義務與修法動向。
- Part 2：表後儲能應用效益，整理降容防超約、時間電價套利、需量反應與電力交易等業主簡報內容。
- Part 3：ESS 專案初步配置問卷，以 Google 表單風格呈現，送出時會開啟郵件草稿寄送至 `davidyeh51@gmail.com`。

## 檔案

- `index.html`：完整簡報與問卷頁面。
- `assets/bess-hero-small.jpg`：首頁主視覺。

## 使用方式

直接開啟 `index.html` 即可瀏覽。發布到 GitHub Pages 後，可把 Pages 網址分享給客戶或業務。

## 注意

問卷採靜態網站相容的 `mailto:` 方式提交，不會把資料存到伺服器。若需要像真正 Google 表單一樣集中收件、匯出試算表，後續可改接 Google Forms、Apps Script 或其他表單後端。
